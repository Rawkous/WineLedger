About WineLedger

WineLedger is a teaching and research-oriented view of a wine supply chain: synthetic events are recorded on a lightweight chain and mapped to generative visuals so provenance and flow are tangible.

Project info

REST API exposes /chain and /simulate-once; live updates stream over WebSocket /ws. The canvas is driven by Hydra from parameters derived from each block.
